◆ PlanThread()

void roborts_global_planner::GlobalPlannerNode::PlanThread ( )
private

Plan thread which mainly include a planning cycle process:

  1. Get the current start pose and goal pose, validate and transform them according to frame of costmap
  2. Make a plan using the selected algorithm
  3. If success, check the completion if the current pose is close to the goal. If completed, no need of planning and jump out of the cycle!
  4. If not succeed, check if it reaches the max retries. if so, no need of planning and jump out of the cycle!
  5. If planning still needed, go to next cycle and wait for a duration which the global planning frequency decides.
